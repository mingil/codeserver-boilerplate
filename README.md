# 🏥 DTx Empire: AI Research Platform (v3.1)

**Global Digital Therapeutics (DTx) 개발을 위한 완전 자동화 연구소**

## ⚡ 주요 기능
1. **자가 복구 (Self-Healing)**: \`./maintenance/repair.sh\` 실행 시 터미널, 권한, 패키지가 자동 복구됩니다.
2. **영속성 (Persistence)**: 리빌드 후에도 확장 프로그램이 자동 재설치됩니다.
3. **스마트 터미널**: Zsh 테마 충돌(경로 고정 문제) 방지 패치 적용 완료.

## 🛠 사용 가이드
* **시스템이 이상할 때**: 터미널에서 \`./maintenance/repair.sh\` 실행.
* **초기 설정 변경**: \`install.sh\` 파일 수정.
