# 📘 DTx Empire: 명령어 가이드 (Cheat Sheet)

이 문서는 터미널이 낯선 CEO를 위한 **원클릭 명령어 설명서**입니다.
VS Code 메뉴의 **[터미널] -> [작업 실행...]** 에서 아래 항목들을 마우스로 선택하세요.

---

### 🛡️ 유지보수 (Maintenance)

| 메뉴 이름 (Label) | 역할 | 언제 쓰나요? |
| :--- | :--- | :--- |
| **✅ 시스템 상태 점검** | `check.sh` 실행 | 접속하자마자, 혹은 뭔가 찜찜할 때 |
| **🚑 시스템 자가 복구** | `repair.sh` 실행 | 빨간 에러가 뜨거나 권한 문제가 생겼을 때 |

### 🐳 도커 & 서버 (Docker)

| 메뉴 이름 (Label) | 역할 | 언제 쓰나요? |
| :--- | :--- | :--- |
| **🐳 도커 상태 확인** | `docker ps` | 내 서버들이 잘 살아있는지 목록을 보고 싶을 때 |
| **💾 Git 백업** | `git push` | 퇴근하기 전, 작업 내용을 클라우드에 저장할 때 |

---

### 🤖 AI 비서 활용법 (Codeium)

* **코드 추천**: 코드를 치다가 회색 글씨가 나오면 **`Tab`** 키를 눌러 수락하세요.
* **AI 채팅**: 왼쪽 사이드바의 Codeium 아이콘을 누르고 "이 함수가 뭐 하는 거야?"라고 물어보세요.

---
**Tip**: 이 파일을 열어두고 `Ctrl + Shift + V`를 누르면 예쁜 미리보기 화면으로 볼 수 있습니다.
